# ☕ Coffeeshop Nongki

Project Pemrograman Web 2 - Kelompok 1

---

## 📌 Deskripsi
Aplikasi web Coffeeshop Nongki dibuat menggunakan Laravel untuk mengelola:
- Menu kopi & makanan
- Sistem login & register (Laravel Breeze)
- Dashboard user

---

## ⚙️ Teknologi
- Laravel 10
- Laravel Breeze (Auth)
- MySQL
- Vite / Node.js
